Spree Commerce

Spree Commerce (also known as Spree) is an open source e-commerce solution based on Ruby on Rails. It was created by Sean Schofield in 2007 and has since had over 740 contributors.
On July 1, 2011, Spree received $1.5 million in seed funding from AOL and True Ventures. On February 25, 2014 Spree raised an additional $5M in Series A funding led by Thrive Capital. Also participating were Vegas Tech Fund (led by Zappos CEO Tony Hsieh), Red Swan (led by Bonobos CEO Andy Dunn) as well as existing investors True Ventures and AOL Ventures.
Companies using Spree include Chipotle Mexican Grill (NYSE: CMG), Second Life, Bonobos (apparel), Fortnum and Mason.
Spree has been downloaded 429,501 times as of January 29, 2016.
On September 21, 2015 it was acquired by First Data
After the First Data acquisition, developers from Spark Solutions and VinSol now maintain the Spree Commerce Open Source project.
